# Vim 插件索引

通过 `install_plugins.sh` 安装，使用 Vim 8 原生包管理（`~/.vim/pack/vendor/start/`）。

---

## nerdtree — 文件树浏览器

在 Vim 侧边栏显示目录树，方便浏览和打开文件。

| 快捷键 | 效果 |
|--------|------|
| `<Space>n` | 打开/关闭文件树 |
| `<Space>f` | 在文件树中定位当前文件 |

文件树内操作：
- `o` / `<Enter>` — 打开文件或展开目录
- `t` — 在新 tab 打开
- `s` — 垂直分屏打开
- `m` — 呼出菜单（新建、删除、重命名文件等）
- `q` — 关闭文件树

---

## vim-commentary — 快捷注释

根据文件类型自动选择注释符号（`.rs` 用 `//`，`.py` 用 `#` 等）。

| 快捷键 | 效果 |
|--------|------|
| `gcc` | 注释/取消注释当前行 |
| `gc` + 动作 | 注释一个范围，如 `gcap` 注释整段，`gc3j` 注释当前行及下面 3 行 |
| 可视模式选中后 `gc` | 批量注释/取消选中行 |

---

## vim-oscyank — SSH 远程剪贴板同步

通过 OSC 52 转义序列将 Vim 中复制的内容同步到本地终端剪贴板，解决 SSH 环境下无法共享剪贴板的问题。

vimrc 中已配置：在 SSH 环境下，每次 yank（`y`）自动同步，无需手动操作。

---

## auto-pairs — 括号自动配对

输入左括号时自动补全右括号，光标停在中间。删除左括号时自动删除配对的右括号。

支持的配对符号：`()` `[]` `{}` `""` `''` `` `` ``

| 场景 | 输入 | 结果 |
|------|------|------|
| 自动补全 | 输入 `(` | 得到 `(⎸)` 光标在中间 |
| 跳过右括号 | 光标在 `⎸)` 时输入 `)` | 光标跳到 `)` 右边，不会重复 |
| 配对删除 | 在 `(⎸)` 中按退格 | 两个括号一起删除 |

无需额外配置，装上即生效。

---

## vim-tmux-clipboard — Vim 与 Tmux 剪贴板互通

在 Tmux 会话中，Vim 的默认寄存器和 Tmux 的粘贴缓冲区自动同步。

- Vim 中 `y` 复制 → Tmux 中 `prefix + ]` 可粘贴
- Tmux 中复制 → Vim 中 `p` 可粘贴

无需额外快捷键，装上即生效。

---

## vim-tmux-focus-events — Tmux 焦点事件修复

让 Tmux 内的 Vim 能正确接收 `FocusGained` / `FocusLost` 事件。

作用：确保切换窗口后 `autoread` 等依赖焦点事件的功能正常工作。是 vim-tmux-clipboard 的依赖。

无需额外配置，装上即生效。

---

## rust.vim — Rust 语言支持

提供 Rust 语法高亮、文件类型检测，以及保存时自动调用 `rustfmt` 格式化代码。

vimrc 中已配置 `let g:rustfmt_autosave = 1`，保存 `.rs` 文件时自动格式化，无需手动操作。
